## 🚀 Instalação

```bash
# Clone o repositório
$ git clone https://github.com/julianapedroso/cenef-web

# Entre na pasta do projeto
$ cd cenef-web

# Instale as dependências
$ npm install / yarn

# Execute a aplicação
$ npm run start / yarn start

# Abra http://localhost:3000 no seu navegador para ver a aplicação!
```
